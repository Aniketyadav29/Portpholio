# 🖼️ Image & Resume Setup Guide

## Files You Need to Add

Follow these steps to add your images and resume to your portfolio:

### 1. **Profile Picture** (Required)
- **Filename:** `aniket-profile.jpg`
- **Size:** Recommended 400x400px or larger (square preferred)
- **Location:** Place in the `Aniketyadav29` folder (same folder as index.html)
- **Format:** JPG, PNG, or WebP
- **Instructions:**
  - Use the image you provided
  - Rename it to exactly: `aniket-profile.jpg`
  - Copy it to this folder

### 2. **Certificate Images** (Required for credentials to display)

#### Kaggle Python Certificate
- **Filename:** `kaggle-certificate.jpg`
- **Size:** Any size works (will auto-fit)
- **Location:** `Aniketyadav29` folder
- **Instructions:**
  - Use the Kaggle certificate image you provided
  - Rename to: `kaggle-certificate.jpg`

#### Lets Upgrade Certificate
- **Filename:** `letsupgrade-certificate.jpg`
- **Size:** Any size works (will auto-fit)
- **Location:** `Aniketyadav29` folder
- **Instructions:**
  - Use the Lets Upgrade - Prompt Engineering certificate
  - Rename to: `letsupgrade-certificate.jpg`

#### Coderush 2.0 Hackathon Certificate
- **Filename:** `coderrush-certificate.jpg`
- **Size:** Any size works (will auto-fit)
- **Location:** `Aniketyadav29` folder
- **Instructions:**
  - Use the Coder Hackathon certificate
  - Rename to: `coderrush-certificate.jpg`

### 3. **Resume PDF** (Required for download button)
- **Filename:** `resume.pdf`
- **Location:** `Aniketyadav29` folder (same as index.html)
- **Instructions:**
  - Copy your `resume.pdf` from `C:\Users\Dell\Downloads\`
  - Place it in the `Aniketyadav29` folder
  - When users click "Download Resume", it will download as "Aniket_Yadav_Resume.pdf"

---

## 💡 Pro Tips

- Images should be compressed for faster loading (use tools like TinyPNG)
- For profile picture, use a high-quality professional headshot
- Certificates look best when they're clear and not too compressed
- Make sure resume.pdf is up-to-date before sharing

---

## 🔗 Need Help?

If images don't show:
1. Check filenames are EXACTLY as listed above
2. Make sure files are in the correct folder
3. Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R on Mac)
4. Check browser console (F12) for any errors
