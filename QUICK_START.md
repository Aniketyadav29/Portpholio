## 🎨 Easy Customization Examples

### Change Your Name
**FIND:**
```html
<h1 class="hero-title">Aniket Yadav</h1>
```
**REPLACE WITH:**
```html
<h1 class="hero-title">John Doe</h1>
```

### Update Your Title
**FIND:**
```html
<span class="highlight">Software & Backend</span> Developer
```
**REPLACE WITH:**
```html
<span class="highlight">Full Stack</span> Developer
```

### Add Your Email
**FIND:**
```html
<a href="mailto:anikety7905@email.com"
```
**REPLACE WITH:**
```html
<a href="mailto:yourname@gmail.com"
```

### Change Primary Color
**FIND (in styles.css):**
```css
--accent-gold: #ffa500;
```
**REPLACE WITH:**
```css
--accent-gold: #00d4ff;  /* For cyan */
--accent-gold: #ff006e;  /* For pink */
--accent-gold: #7c3aed;  /* For purple */
```
